# DTMF (Photo Manager)

## 介绍 (Introduction)
DTMF 是一款类似 Tinder 风格的 Android 快捷照片管理应用。它可以帮助你快速清理手机相册：通过简单的左右滑动手势，轻松将不需要的照片加入删除暂存区，保留喜欢的照片，最后在暂存区进行集中批量删除。

## 怎么使用 (Usage)
1. **浏览照片**：打开应用后，会展示相册中的照片。
2. **左滑删除**（Swipe Left）：将当前照片移动到**删除暂存区**。
3. **右滑保留**（Swipe Right）：保留照片，跳过当前图片。
4. **进入暂存区**：点击“Proceed”按钮进入删除暂存区（Delete Staging Area）。
5. **集中删除**：在暂存区中，勾选需要确认删除的照片，进行最终的永久批量删除操作。

## 怎么安装 (Installation)
1. 克隆本项目代码到本地。
2. 使用 **Android Studio** 打开项目文件夹 (`wish-dtmf`)。
3. 等待 Gradle 同步所有依赖项（项目使用 Jetpack Compose 等现代 Android 技术栈编写）。
4. 连接 Android 真机或启动模拟器（最低支持 Android 11 / API 30及以上）。
5. 点击 Android Studio 顶部的 **Run** 按钮（绿色三角形）编译并运行应用。
